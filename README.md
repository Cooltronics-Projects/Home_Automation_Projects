# 🏠 Home Automation Projects

A collection of DIY home automation projects built with ESP32, Arduino, and other microcontrollers. Each project lives in its own folder with full source code, wiring diagrams, and documentation.

---

## 📺 About This Repo

This repository is the companion to the **[Cooltronics Projects](https://www.youtube.com/@CooltronicsProjects)** YouTube channel — where we build real, working home automation systems using affordable components available in India.

Every project here is:
- ✅ Beginner-friendly
- ✅ Built and tested on real hardware
- ✅ Fully documented with code + wiring

## 🛠️ Common Components Used

- ESP32 / ESP8266
- Arduino Uno / Nano
- Relay Modules
- DHT11 / DHT22 Temperature & Humidity Sensors
- OLED Displays (SSD1306)
- IR Sensors / PIR Motion Sensors
- MQTT / Blynk / Home Assistant

---

## 🚀 Getting Started

1. Clone this repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Home_Automation_Projects.git
   ```
2. Navigate into any project folder
3. Open the project's `README.md` for wiring + setup instructions
4. Upload the code using Arduino IDE or PlatformIO

---

## 📌 Requirements

- [Arduino IDE](https://www.arduino.cc/en/software) or [PlatformIO](https://platformio.org/)
- ESP32 board package installed
- Required libraries listed in each project's `README.md`
